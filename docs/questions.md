# Dudas del Proyecto: Island Jam Copilot 2026 🏝️

Para asegurar que el "Island Jam Copilot" sea la herramienta definitiva durante el crunch, necesito aclarar los siguientes puntos:

1. **Autenticación de Equipos:** ¿La herramienta será abierta para cualquier participante del evento o requiere un sistema de login por equipo/mesa para persistir sus "Logs de Decisión"?
2. **Integración Multimodal (Gemini 3.0 Flash):** Para la subida de fotos de cuadernos/bocetos, ¿prefieres integración directa con un bucket de S3/Google Cloud Storage o una gestión temporal en base64 para prototipado rápido?
3. **Motores de Juego:** El PRD menciona Unity, Unreal y Godot. ¿Deberíamos incluir soporte para motores web (ej. Three.js/Phaser) dado que es una herramienta web?
4. **API Key de Gemini:** ¿Ya cuentas con una clave configurada en el entorno o prefieres que implemente un "Mock Mode" para desarrollo local inicial?
5. **Horario Real:** ¿Los hitos del Roadmap deben ser configurables o los dejo "hardcoded" según el horario oficial de la Gran Canaria Game Island JAM 2026?
6. **Next.js 16:** ¿Hay alguna feature específica de la v16 (como el nuevo sistema de caches o el proxy) que quieras que explote al máximo para la baja latencia?
